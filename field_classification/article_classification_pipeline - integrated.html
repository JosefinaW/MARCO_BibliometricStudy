Overview

This pipeline classifies scientific articles into disciplinary clusters and fields based on title, abstract, and journal title. The classification schema is adapted from the German Federal Statistical Office (Fächersystematik) with modifications.

Key Design Decisions

  1. Two-step classification: First assign to a Subject Area, then (if area has >100 papers) assign to a Subject Field within that area
  2. Journal-title-first classification: Classification relies primarily on journal titles to minimize interpretation; article content (title and abstract) is used for multidisciplinary journals
  3. Secondary field for multidisciplinary work: Each article receives one primary classification, with an optional secondary Subject Area for multidisciplinary articles
  4. Hierarchical flexibility: Subject Areas provide broad categories; Subject Fields provide granularity where needed

Modifications from Original Stabu Schema

  • Merged all theology into “Religious Studies & Theology”

  • Merged Media Studies into Communication Studies

  • Added: Marketing, Behavioural/Experimental Economics, Experimental Philosophy

  • Tightened definition of Occupational/Organisational/Business Psychology

  • Changed General and Cognitive Psychology to Cognitive Psychology, to distinguish from Psychology (General)

  • Pull Science Studies from Philosophy into its own cluster to allow for correct classification of metascience papers

  • Added specific rules for metascience, consumer behaviour, and JDM research

clean_json_text <- function(x) {
  x <- as.character(x)
  # Build replacements safely
  bytes <- c(0x91, 0x92, 0x93, 0x94, 0x96, 0x97, 0x85, 0xD5, 0xCA)
  repl <- c("'", "'", '"', '"', "-", "-", "...", "'", "")
  names(repl) <- vapply(
    bytes,
    function(b) rawToChar(as.raw(b), multiple = FALSE),
    FUN.VALUE = character(1)
  )
  # Replace byte patterns
  for (bad in names(repl)) {
    x <- gsub(bad, repl[[bad]], x, fixed = TRUE, useBytes = TRUE)
  }
  # Drop control characters except newline and tab
  x <- stringi::stri_replace_all_regex(
    x,
    "[\\p{Cc}&&[^\\n\\t]]",
    ""
  )
  # Final UTF-8 normalisation
  iconv(x, from = "", to = "UTF-8", sub = "")  
}

# Helper function for missing value handling
`%||%` <- function(x, y) {
  if (is.null(x)) return(y)
  if (is.character(x) && length(x) == 1 && is.na(x)) return(y)
  if (length(x) == 0) return(y)
  x
}

validate_cluster_schema <- function(x) {
  if (!is.list(x) || is.null(x$clusters)) {
    stop("cluster_schema must be a list with element 'clusters'.")
  }

  allowed_top <- c("version", "description", "clusters")
  extra_top <- setdiff(names(x), allowed_top)
  if (length(extra_top) > 0) {
    warning(
      "Ignoring unsupported top-level fields in cluster_schema: ",
      paste(extra_top, collapse = ", ")
    )
  }

  x$clusters <- lapply(x$clusters, function(cl) {
    if (is.null(cl$id) || is.null(cl$label)) {
      stop("Each cluster must have 'id' and 'label'.")
    }
    allowed <- c("id", "label", "notes")
    extra <- setdiff(names(cl), allowed)
    if (length(extra) > 0) {
      warning(
        "Ignoring unsupported fields in cluster '", cl$id, "': ",
        paste(extra, collapse = ", ")
      )
    }
    # Don't add empty fields - code handles missing fields with %||% operator
    cl
  })

  x
}

Cluster Classification Prompt Template

# Reference template - used with glue(cluster_prompt_template, cluster_schema = cluster_schema_json)
cluster_prompt_template <- ('
You classify scientific articles into discipline clusters (Subject Areas).

Use ONLY the clusters defined in the JSON array below.
Classification should rely primarily on the journal title, as it typically indicates the disciplinary field.
Only when the journal is clearly multidisciplinary or interdisciplinary (e.g., Nature, Science, PLOS ONE, PNAS),
use the article title and abstract to determine the best-fitting cluster.
Never invent new clusters; always return one of the existing id values.

For multidisciplinary articles, assign one primary cluster that best fits the core contribution.
If the article genuinely spans two disciplines, also assign a secondary cluster.

Key distinctions:
- Psychology vs Sociology: Individual processes vs social structure/institutions
- Psychology vs Economics: Psychological theory vs economic framing (utility, incentives, markets)
- Education vs Teaching: Educational systems vs teaching methods in a discipline
- Media vs Political Science: Media/communication vs political processes/institutions
- Metascience: Only for science in general, not discipline-specific methods
- Methods papers: Belong to the discipline being studied

AVAILABLE CLUSTERS (JSON array):
{cluster_schema}

You will receive articles as JSON with: article_title, article_abstract, journal_title

Respond with a JSON object containing:
- "cluster_id": the id of the primary matching cluster (string)
- "confidence": your confidence from 0 to 1 (number)
- "secondary_cluster_id": the id of a secondary cluster if the article is multidisciplinary (string or null)
- "secondary_confidence": confidence for the secondary cluster, if applicable (number or null)
')

Psychology Fields Classification Prompt Template

# Reference template - used with glue(psychology_field_prompt_template, psychology_fields_schema = psychology_fields_json)
psychology_field_prompt_template <- '
You classify psychology articles into a single primary psychology subfield (Subject Field).

Use ONLY the fields defined in the JSON array below.
Classification should rely primarily on the journal title where it indicates a specific subfield.
For general psychology journals, use the article title and abstract to determine the best-fitting subfield.
If interdisciplinary, pick the subfield that best fits the core psychological contribution.
Never invent new fields; always return one of the existing id values.

Key distinctions:
- Psychology (General): Last resort only when no single subfield clearly dominates
- Occupational Psychology: Requires BOTH work-specific constructs AND theoretical claims about work
  - Workplace as setting only → classify by underlying theory (e.g., prejudice reduction → Social)
  - Work-specific constructs (leadership, burnout, job satisfaction) → Occupational
- Educational Psychology: Learning/motivation/assessment in educational settings only
- Methods: Primary contribution must be methodological (new measure, technique, or meta-analytic method)

AVAILABLE FIELDS (JSON array):
{psychology_fields_schema}

You will receive articles as JSON with: article_title, article_abstract, journal_title

For multidisciplinary articles that genuinely span two subfields (e.g. a judgment/decision-making paper with a strong social-identity component), assign one primary field that best fits the core contribution and also assign a secondary field. Otherwise leave the secondary fields null.

Respond with a JSON object containing:
- "field_id": the id of the primary matching field (string)
- "confidence": your confidence from 0 to 1 (number)
- "secondary_field_id": the id of a secondary field if the article genuinely spans two subfields (string or null)
- "secondary_confidence": confidence for the secondary field, if applicable (number or null)
'

# Create cluster classification chat with provider flexibility
create_cluster_chat <- function(
  model = "openai/gpt-oss-120b",
  provider = "deepinfra",
  api_args = list(temperature = 0)
) {
  system_prompt <- glue(cluster_prompt_template,
                        cluster_schema = cluster_schema_json)

  if (provider == "deepinfra") {
    # Use chat_openai_compatible for DeepInfra (uses /chat/completions endpoint;
    # chat_openai in ellmer >= 0.4.0 uses /responses which DeepInfra doesn't support)
    deepinfra_args <- modifyList(api_args, list(
      prompt_cache_key = "cluster_classification"
    ))
    chat_openai_compatible(
      base_url = "https://api.deepinfra.com/v1/openai",
      name = "deepinfra",
      system_prompt = system_prompt,
      model = model,
      credentials = function() Sys.getenv("DEEPINFRA_API_KEY"),
      api_args = deepinfra_args,
      echo = "none"
    )
  } else if (provider == "openai") {
    chat_openai(
      system_prompt = system_prompt,
      model = model,
      api_args = api_args,
      echo = "none"
    )
  } else if (provider == "anthropic") {
    chat_anthropic(
      system_prompt = system_prompt,
      model = model,
      api_args = api_args,
      echo = "none"
    )
  } else {
    stop("Provider must be 'deepinfra', 'openai', or 'anthropic'")
  }
}

# Extract classification results and reasoning from parallel_chat responses
extract_cluster_results <- function(chats) {
  map_dfr(chats, function(chat) {
    turns <- chat$get_turns()
    assistant_turn <- turns[[2]]

    # Parse JSON content
    content <- assistant_turn@text
    parsed <- tryCatch(
      fromJSON(content),
      error = function(e) list(cluster_id = NA_character_, confidence = NA_real_,
                               secondary_cluster_id = NA_character_, secondary_confidence = NA_real_)
    )

    # Extract reasoning_content from raw response
    reasoning <- tryCatch(
      assistant_turn@json$choices[[1]]$message$reasoning_content %||% NA_character_,
      error = function(e) NA_character_
    )

    tibble(
      cluster_id = parsed$cluster_id %||% NA_character_,
      confidence = parsed$confidence %||% NA_real_,
      secondary_cluster_id = parsed$secondary_cluster_id %||% NA_character_,
      secondary_confidence = parsed$secondary_confidence %||% NA_real_,
      reasoning = reasoning
    )
  })
}

classify_articles_batch <- function(
  articles_df,
  title_col = "title",
  abstract_col = "abstract",
  journal_col = "journal",
  model = "openai/gpt-oss-120b",
  provider = "deepinfra"
) {
  # Create chat for cluster classification
  cluster_chat <- create_cluster_chat(
    model = model,
    provider = provider,
    api_args = list(temperature = 0)
  )

  # Prepare prompts for batch processing
  articles_list <- articles_df |>
    rowwise() |>
    reframe(
      article = toJSON(
        list(
          article_title = .data[[title_col]] %||% "",
          article_abstract = .data[[abstract_col]] %||% "",
          journal_title = .data[[journal_col]] %||% ""
        ),
        auto_unbox = TRUE
      )
    ) |>
    pull(article) |>
    as.character()  |>
    clean_json_text() |>
    as.list()

  # Batch classify clusters using parallel processing
  # Use parallel_chat (not parallel_chat_structured) to access reasoning_content
  cat("Classifying articles into clusters (parallel processing)...\n")
  chat_results <- parallel_chat(
    chat = cluster_chat,
    prompts = articles_list
  )

  # Extract structured results and reasoning
  cluster_results_df <- extract_cluster_results(chat_results)

  # Combine results with original data
  articles_df |>
    bind_cols(cluster_results_df)
}

Field Classification for Large Clusters

Code 
classify_cluster_fields <- function(
  cluster_results,
  cluster_breakdowns,
  title_col = "title",
  abstract_col = "abstract",
  journal_col = "journal",
  model = "openai/gpt-oss-120b",
  provider = "deepinfra",
  field_threshold = 100
) {
  # Count articles per cluster — include both primary and secondary assignments
  cluster_counts <- bind_rows(
    cluster_results |> transmute(cluster_id = cluster_id),
    cluster_results |> transmute(cluster_id = secondary_cluster_id)
  ) |>
    filter(!is.na(cluster_id)) |>
    count(cluster_id) |>
    arrange(desc(n))

  cat("\n=== CLUSTERS REQUIRING FIELD CLASSIFICATION (primary + secondary) ===\n")
  clusters_above_threshold <- cluster_counts |>
    filter(n >= field_threshold)

  if (nrow(clusters_above_threshold) > 0) {
    print(clusters_above_threshold)
  } else {
    cat("No clusters with >= ", field_threshold, " articles\n")
    return(cluster_results)
  }

  # Initialise field columns, one set per cluster with a schema, so articles
  # where the cluster appears as secondary can be stored without clobbering
  # their primary-cluster fields.
  for (cl_id in names(cluster_breakdowns)) {
    field_col  <- paste0(cl_id, "_field_id")
    conf_col   <- paste0(cl_id, "_field_confidence")
    sec_col    <- paste0(cl_id, "_secondary_field_id")
    sec_conf   <- paste0(cl_id, "_secondary_field_confidence")
    reas_col   <- paste0(cl_id, "_field_reasoning")
    if (is.null(cluster_results[[field_col]])) cluster_results[[field_col]]  <- NA_character_
    if (is.null(cluster_results[[conf_col]]))  cluster_results[[conf_col]]   <- NA_real_
    if (is.null(cluster_results[[sec_col]]))   cluster_results[[sec_col]]    <- NA_character_
    if (is.null(cluster_results[[sec_conf]]))  cluster_results[[sec_conf]]   <- NA_real_
    if (is.null(cluster_results[[reas_col]]))  cluster_results[[reas_col]]   <- NA_character_
  }

  for (target_cluster in clusters_above_threshold$cluster_id) {
    if (is.na(target_cluster) || !(target_cluster %in% names(cluster_breakdowns))) {
      cat(sprintf("Note: No field schema available for cluster '%s'\n", target_cluster))
      next
    }

    config <- cluster_breakdowns[[target_cluster]]

    # Articles where target_cluster appears as primary OR secondary
    is_target <- (cluster_results$cluster_id == target_cluster &
                    !is.na(cluster_results$cluster_id)) |
      (cluster_results$secondary_cluster_id == target_cluster &
         !is.na(cluster_results$secondary_cluster_id))
    articles_idx <- which(is_target)
    n_articles <- length(articles_idx)

    cat(sprintf(
      "\nClassifying %d %s articles (primary or secondary) into fields...\n",
      n_articles, target_cluster
    ))
    if (n_articles == 0) next

    field_chat <- config$chat_constructor(
      model = model,
      provider = provider,
      api_args = list(temperature = 0)
    )

    field_prompts <- cluster_results[articles_idx, ] |>
      rowwise() |>
      reframe(
        article = toJSON(
          list(
            article_title = .data[[title_col]] %||% "",
            article_abstract = .data[[abstract_col]] %||% "",
            journal_title = .data[[journal_col]] %||% ""
          ),
          auto_unbox = TRUE
        )
      ) |>
      pull(article) |>
      as.character() |>
      clean_json_text() |>
      as.list()

    field_chat_results <- parallel_chat(
      chat = field_chat,
      prompts = field_prompts
    )
    field_results_df <- extract_field_results(field_chat_results)

    # Write into the per-cluster columns
    cluster_results[[paste0(target_cluster, "_field_id")]][articles_idx] <-
      field_results_df$field_id
    cluster_results[[paste0(target_cluster, "_field_confidence")]][articles_idx] <-
      field_results_df$field_confidence
    cluster_results[[paste0(target_cluster, "_secondary_field_id")]][articles_idx] <-
      field_results_df$secondary_field_id
    cluster_results[[paste0(target_cluster, "_secondary_field_confidence")]][articles_idx] <-
      field_results_df$secondary_field_confidence
    cluster_results[[paste0(target_cluster, "_field_reasoning")]][articles_idx] <-
      field_results_df$field_reasoning
  }

  # Also surface the primary cluster's field result as unprefixed `field_id`
  # for convenience (back-compat with the old column name).
  cluster_results <- cluster_results |>
    mutate(
      field_id = NA_character_,
      field_confidence = NA_real_,
      secondary_field_id = NA_character_,
      secondary_field_confidence = NA_real_,
      field_reasoning = NA_character_
    )
  for (cl_id in names(cluster_breakdowns)) {
    has_primary <- which(cluster_results$cluster_id == cl_id &
                           !is.na(cluster_results$cluster_id))
    if (length(has_primary) > 0) {
      cluster_results$field_id[has_primary] <-
        cluster_results[[paste0(cl_id, "_field_id")]][has_primary]
      cluster_results$field_confidence[has_primary] <-
        cluster_results[[paste0(cl_id, "_field_confidence")]][has_primary]
      cluster_results$secondary_field_id[has_primary] <-
        cluster_results[[paste0(cl_id, "_secondary_field_id")]][has_primary]
      cluster_results$secondary_field_confidence[has_primary] <-
        cluster_results[[paste0(cl_id, "_secondary_field_confidence")]][has_primary]
      cluster_results$field_reasoning[has_primary] <-
        cluster_results[[paste0(cl_id, "_field_reasoning")]][has_primary]
    }
  }

  cluster_results
}

## Overview

This pipeline classifies scientific articles into disciplinary clusters and fields based on title, abstract, and journal title. The classification schema is adapted from the German Federal Statistical Office (Fächersystematik) with modifications.

### Key Design Decisions

1. **Two-step classification**: First assign to a Subject Area, then (if area has >100 papers) assign to a Subject Field within that area
2. **Journal-title-first classification**: Classification relies primarily on journal titles to minimize interpretation; article content (title and abstract) is used for multidisciplinary journals
3. **Secondary field for multidisciplinary work**: Each article receives one primary classification, with an optional secondary Subject Area for multidisciplinary articles
4. **Hierarchical flexibility**: Subject Areas provide broad categories; Subject Fields provide granularity where needed

### Modifications from Original Stabu Schema

- Merged all theology into "Religious Studies & Theology"
- Merged Media Studies into Communication Studies
- Added: Marketing, Behavioural/Experimental Economics, Experimental Philosophy
- Tightened definition of Occupational/Organisational/Business Psychology
- Changed General and Cognitive Psychology to Cognitive Psychology, to distinguish from Psychology (General)
- Pull Science Studies from Philosophy into its own cluster to allow for correct classification of metascience papers

- Added specific rules for metascience, consumer behaviour, and JDM research

    function(b) rawToChar(as.raw(b), multiple = FALSE),
    FUN.VALUE = character(1)
  )
  # Replace byte patterns
  for (bad in names(repl)) {
    x <- gsub(bad, repl[[bad]], x, fixed = TRUE, useBytes = TRUE)

  )
  # Final UTF-8 normalisation
  iconv(x, from = "", to = "UTF-8", sub = "")  
}

validate_cluster_schema <- function(x) {
  if (!is.list(x) || is.null(x$clusters)) {
    stop("cluster_schema must be a list with element 'clusters'.")
  }

  allowed_top <- c("version", "description", "clusters")
  extra_top <- setdiff(names(x), allowed_top)
  if (length(extra_top) > 0) {
    warning(
      "Ignoring unsupported top-level fields in cluster_schema: ",
      paste(extra_top, collapse = ", ")
    )
  }

  x$clusters <- lapply(x$clusters, function(cl) {
    if (is.null(cl$id) || is.null(cl$label)) {
      stop("Each cluster must have 'id' and 'label'.")
    }
    allowed <- c("id", "label", "notes")
    extra <- setdiff(names(cl), allowed)
    if (length(extra) > 0) {
      warning(
        "Ignoring unsupported fields in cluster '", cl$id, "': ",
        paste(extra, collapse = ", ")
      )
    }
    # Don't add empty fields - code handles missing fields with %||% operator
    cl
  })

  x
}

create_cluster_chat <- function(
  model = "openai/gpt-oss-120b",
  provider = "deepinfra",
  api_args = list(temperature = 0)
) {
  system_prompt <- glue(cluster_prompt_template,
                        cluster_schema = cluster_schema_json)

  if (provider == "deepinfra") {
    # Use chat_openai_compatible for DeepInfra (uses /chat/completions endpoint;
    # chat_openai in ellmer >= 0.4.0 uses /responses which DeepInfra doesn't support)
    deepinfra_args <- modifyList(api_args, list(
      prompt_cache_key = "cluster_classification"
    ))
    chat_openai_compatible(
      base_url = "https://api.deepinfra.com/v1/openai",
      name = "deepinfra",
      system_prompt = system_prompt,
      model = model,
      credentials = function() Sys.getenv("DEEPINFRA_API_KEY"),
      api_args = deepinfra_args,
      echo = "none"
    )
  } else if (provider == "openai") {
    chat_openai(
      system_prompt = system_prompt,
      model = model,
      api_args = api_args,
      echo = "none"
    )
  } else if (provider == "anthropic") {
    chat_anthropic(
      system_prompt = system_prompt,
      model = model,
      api_args = api_args,
      echo = "none"
    )
  } else {
    stop("Provider must be 'deepinfra', 'openai', or 'anthropic'")

classify_articles_batch <- function(
  articles_df,
  title_col = "title",
  abstract_col = "abstract",
  journal_col = "journal",
  model = "openai/gpt-oss-120b",
  provider = "deepinfra"
) {
  # Create chat for cluster classification
  cluster_chat <- create_cluster_chat(
    model = model,
    provider = provider,
    api_args = list(temperature = 0)
  )

  # Prepare prompts for batch processing
  articles_list <- articles_df |>
    rowwise() |>
    reframe(
      article = toJSON(
        list(
          article_title = .data[[title_col]] %||% "",
          article_abstract = .data[[abstract_col]] %||% "",
          journal_title = .data[[journal_col]] %||% ""
        ),
        auto_unbox = TRUE
      )
    ) |>
    pull(article) |>
    as.character()  |>
    clean_json_text() |>
    as.list()

  # Batch classify clusters using parallel processing
  # Use parallel_chat (not parallel_chat_structured) to access reasoning_content
  cat("Classifying articles into clusters (parallel processing)...\n")
  chat_results <- parallel_chat(
    chat = cluster_chat,
    prompts = articles_list
  )

  # Extract structured results and reasoning
  cluster_results_df <- extract_cluster_results(chat_results)

  # Combine results with original data
  articles_df |>
    bind_cols(cluster_results_df)
}

## Field Classification for Large Clusters

```{r}
#| label: field-processing-parallel

classify_cluster_fields <- function(
  cluster_results,
  cluster_breakdowns,
  title_col = "title",
  abstract_col = "abstract",
  journal_col = "journal",
  model = "openai/gpt-oss-120b",
  provider = "deepinfra",
  field_threshold = 100
) {
  # Count articles per cluster — include both primary and secondary assignments
  cluster_counts <- bind_rows(
    cluster_results |> transmute(cluster_id = cluster_id),
    cluster_results |> transmute(cluster_id = secondary_cluster_id)
  ) |>
    filter(!is.na(cluster_id)) |>
    count(cluster_id) |>
    arrange(desc(n))

  cat("\n=== CLUSTERS REQUIRING FIELD CLASSIFICATION (primary + secondary) ===\n")
  clusters_above_threshold <- cluster_counts |>
    filter(n >= field_threshold)

  if (nrow(clusters_above_threshold) > 0) {
    print(clusters_above_threshold)
  } else {
    cat("No clusters with >= ", field_threshold, " articles\n")
    return(cluster_results)
  }

  # Initialise field columns, one set per cluster with a schema, so articles
  # where the cluster appears as secondary can be stored without clobbering
  # their primary-cluster fields.
  for (cl_id in names(cluster_breakdowns)) {
    field_col  <- paste0(cl_id, "_field_id")
    conf_col   <- paste0(cl_id, "_field_confidence")
    sec_col    <- paste0(cl_id, "_secondary_field_id")
    sec_conf   <- paste0(cl_id, "_secondary_field_confidence")
    reas_col   <- paste0(cl_id, "_field_reasoning")
    if (is.null(cluster_results[[field_col]])) cluster_results[[field_col]]  <- NA_character_
    if (is.null(cluster_results[[conf_col]]))  cluster_results[[conf_col]]   <- NA_real_
    if (is.null(cluster_results[[sec_col]]))   cluster_results[[sec_col]]    <- NA_character_
    if (is.null(cluster_results[[sec_conf]]))  cluster_results[[sec_conf]]   <- NA_real_
    if (is.null(cluster_results[[reas_col]]))  cluster_results[[reas_col]]   <- NA_character_
  }

  for (target_cluster in clusters_above_threshold$cluster_id) {
    if (is.na(target_cluster) || !(target_cluster %in% names(cluster_breakdowns))) {
      cat(sprintf("Note: No field schema available for cluster '%s'\n", target_cluster))
      next
    }

    config <- cluster_breakdowns[[target_cluster]]

    # Articles where target_cluster appears as primary OR secondary
    is_target <- (cluster_results$cluster_id == target_cluster &
                    !is.na(cluster_results$cluster_id)) |
      (cluster_results$secondary_cluster_id == target_cluster &
         !is.na(cluster_results$secondary_cluster_id))
    articles_idx <- which(is_target)
    n_articles <- length(articles_idx)

    cat(sprintf(
      "\nClassifying %d %s articles (primary or secondary) into fields...\n",
      n_articles, target_cluster
    ))
    if (n_articles == 0) next

    field_chat <- config$chat_constructor(
      model = model,
      provider = provider,
      api_args = list(temperature = 0)
    )

    field_prompts <- cluster_results[articles_idx, ] |>
      rowwise() |>
      reframe(
        article = toJSON(
          list(
            article_title = .data[[title_col]] %||% "",
            article_abstract = .data[[abstract_col]] %||% "",
            journal_title = .data[[journal_col]] %||% ""
          ),
          auto_unbox = TRUE
        )
      ) |>
      pull(article) |>
      as.character() |>
      clean_json_text() |>
      as.list()

    field_chat_results <- parallel_chat(
      chat = field_chat,
      prompts = field_prompts
    )
    field_results_df <- extract_field_results(field_chat_results)

    # Write into the per-cluster columns
    cluster_results[[paste0(target_cluster, "_field_id")]][articles_idx] <-
      field_results_df$field_id
    cluster_results[[paste0(target_cluster, "_field_confidence")]][articles_idx] <-
      field_results_df$field_confidence
    cluster_results[[paste0(target_cluster, "_secondary_field_id")]][articles_idx] <-
      field_results_df$secondary_field_id
    cluster_results[[paste0(target_cluster, "_secondary_field_confidence")]][articles_idx] <-
      field_results_df$secondary_field_confidence
    cluster_results[[paste0(target_cluster, "_field_reasoning")]][articles_idx] <-
      field_results_df$field_reasoning
  }

  # Also surface the primary cluster's field result as unprefixed `field_id`
  # for convenience (back-compat with the old column name).
  cluster_results <- cluster_results |>
    mutate(
      field_id = NA_character_,
      field_confidence = NA_real_,
      secondary_field_id = NA_character_,
      secondary_field_confidence = NA_real_,
      field_reasoning = NA_character_
    )
  for (cl_id in names(cluster_breakdowns)) {
    has_primary <- which(cluster_results$cluster_id == cl_id &
                           !is.na(cluster_results$cluster_id))
    if (length(has_primary) > 0) {
      cluster_results$field_id[has_primary] <-
        cluster_results[[paste0(cl_id, "_field_id")]][has_primary]
      cluster_results$field_confidence[has_primary] <-
        cluster_results[[paste0(cl_id, "_field_confidence")]][has_primary]
      cluster_results$secondary_field_id[has_primary] <-
        cluster_results[[paste0(cl_id, "_secondary_field_id")]][has_primary]
      cluster_results$secondary_field_confidence[has_primary] <-
        cluster_results[[paste0(cl_id, "_secondary_field_confidence")]][has_primary]
      cluster_results$field_reasoning[has_primary] <-
        cluster_results[[paste0(cl_id, "_field_reasoning")]][has_primary]
    }
  }

  cluster_results
}
```
